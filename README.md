# imdb_dataset_500

A subset of imdb dataset for 500 movies, created for the klimbB challenge.

## Datasets

For the purpose of the klimbB challenge, we have created smaller subsets of
IMDb datasets. The following tsv files are made available in this repository:

- `name.basics.small.tsv`
- `title.basics.small.tsv`
- `title.principals.small.tsv`
